## 1. 선형대수학

#### 기본 연산
- Add Vector : v+w
- Multiple Vector : cv, dw (c,d : real number scalar)
- Linear Combination : cv + dw
	- sum : 1v + 1w
	- sub : 1v - 1w
	- zero vector : 0v + 0w
	- scalar multiple : cv + dw
