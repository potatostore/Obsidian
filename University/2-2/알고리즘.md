# 중간고사

- 알고리즘 : 모호하지 않고 실행 가능한 단계들의 집합
- 의사코드 : 알고리즘 개발단계에서 자유로운 형식으로 표현하는 체계

#### 표기법
- big-O : 최악의 성능 표기
- big-omega : 최선의 성능 표기
- big-theta : 최악과 최선의 성능 모두 표기

#### 알고리즘
- 순차검색 : 처음부터 끝까지 탐색
- 이진탑색
- divide and conquer approach : 분할 정복
- quick sort : pivot값을 통해 좌우 정렬을 반복

- 교환정렬 : 처음 인덱스 원소를 기준으로 나머지를 순차적으로 비교, 교환을 반복

#### 기타 등등
- 수학적 귀납법 : R(n) = f(n)과 같은 식을 풀 때, R(n+1) = f(n+1)도 같은지 확인하면서 증명
- 등차수열 합공식 : $S_n = \frac{n}{2}(2a_1 + (n-1)d)$
- 등비수열 합공식 : $S_n = \frac{a(r_{n} - 1)}{r-1}$
- 행렬곱 : 내적
- 피보나치 수열 : f(n) = f(n-1) + f(n-2) -> 재귀함수 쓰기
