- 토지, 노동에만 의존하던 공간에 에너지를 도입하여 공간이 갖는 생산력 확대

#### 인터넷 혁명(정보 통신 혁명)
- 정보 기술의 발달로 전자공간 등장
- 사회경제적 활동을 전자공간에서 수행 -> 실체적 공간으로 급성장

#### 유비쿼터스 개념 등장
- 마크와이저의 유비쿼터스 컴퓨팅 개념으로 전자 공간 중심의 정보화 패러다임에 또 다른 대전환

#### 공간 혁명(유비쿼터스 혁명)
- 기존 전자공간과 물리공간의 이분화에서 벗어나 이 둘을 유기적으로 합침 -> 유비쿼터스 공간으로 진화
- 물리 공간의 모든 대상을 연결시키는 통신 대상의 무한 확장 개념
- 기존 컴퓨터 중심적이었던 컴퓨팅 시스템에서 사람-사물-컴퓨터 간의 연계 주장(전자공간과 물리공간의 연계 의미)
- 각종 사물과 물리적 환경 전반(물리공간)에 컴퓨터가 적절히, 효과적으로 삽입되어 보이지 않으면서(투명성), 정보흐름이 이뤄저야 한다고 주장

## 전자공간과 물리공간의 연계 접근 방법
#### 전자공간과 물리공간의 연계에 대한 접근 시각
- 증강현실을 전자공간과 물리공간의 연계로 보는 시각
	- 휴대용 비디오 카메라를 매개체로 현실세계의 이미지를 해석하여 컴퓨터가 인식
- 유비쿼터스 컴퓨팅 개념에 입각한 시각
	- 물리공간 속 수많은 작은 컴퓨터를  통해 사물-사물, 사물-컴퓨터, 컴퓨터-사물 상호작용을 인식
	- 사물-컴퓨터-사람이 네트워크로 연결되어 소통

#### 연계 성립 조건
- 사물-컴퓨터-사람의 네트워크화 차원 : 물리공간에 심어진 작은 컴퓨터가 네트워크를 통해 연결되어야 함
- 물리적 속성의 인식 차원 : 물리공간에 심어진 작은 컴퓨터가 전자공간에서의 응용/서비스 수행에 필요한 물리적 속성을 포착 및 통식, 구동 가능
- 물리적 속성과 전자적 속성 간의 연계 차원 : 물리공간의 속성 상태는 전자적 속성과 상호작용 가능

-> *전자공간과 물리공간 간의 연계 : 네트워크 기반, 물리공간에 대한 상황인식 가능, 물리공간/전자공간 간 양방향 상호작용 가능*

#### 공간과학성
- 물리공간/전자공간의 한계 극복, 차세대 정보 서비스와 응용 개발에 중요한 방향 제시

#### 전자공간/물리공간의 연계에 대한 기본적 매커니즘
- 전자공간/물리공간 속성들이 사용자가 요구하는 서비스 제공 가능하도록 시스템적 상요작용하는 것이 핵심

#### 웹 현실화 기술을 활용한 전자공간과 물리공간의 연계
- 물리공간 3대 요소(사물,사람,컴퓨터)를 웹상에 표현하는 기술
- 사용자의 상황에 기초하여 물리적 속성에 관한 웹 정보 내용의 자동 생성 및 상호 관계성 유지 기술
