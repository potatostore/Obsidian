## 초연결사회
- 다양한 사물이 지능을 갖고 데이터를 생성(유비쿼터스 혁명) -> 데이터들을 연결하는 초연결사회 탄생
- 사람과 사람을 둘러싸고 있는 다양한 사물이 생성하는 데이터 간 연결을 통해 사회 전 분야에 변화와 혁신을 이끄는 사회

#### 등장 배경
1. VAN의 등장(1970년대) : 기존 네트워크를 사용하지 않은 곳에서 네트워크 사용, 컴퓨터 중심의 비즈니스 출현
2. 인터넷 보급(1990년대) : 전자상거래 확대
3. 모바일 보급(2000년대) : 개인에 의한 정보생성 및 새로운 생태계 탄생
4. IoT/AI 진화(현재) :사물인터넷을 통한 초연결사회

#### 6Any
- 기존 3Any(언제,어디서,누구나)에서 3Any(무엇이든, 어떤 서비스든, 어떤 네트워크든)추가

#### 가상물리시스템(CPS)
- 4차 산업혁명 핵심기술 간 상호연결 시 네트워크, 운영체제, 복잡성 등으로 인한 결함 발생
- 결함 극복을 위한 자율제어시스템, CPS 등장
- CPS : 현실세계와 가상세계 간 연결을 통해 현실과 가상 데이터 융합/분석 및 결과 데이터를 현실세계에 환류하는 시스템

- 초연결 : 각종 데이터를 클라우드 상에서 빅데이터로 수집
- 초지능 : 초연결을 통해 얻은 빅테이터를 인공지능과 결합한 빅데이터 처리 기술을 통해 정보 지식화
- 각종 서비스에 사용

- 핵심기술
	1. 컴퓨팅 및 S/W 기술 : 시스템 수준 오류 탐지 및 복구 기술
	2. 통신 기술 : 신뢰성 있고 유연한 네트워크 제공 가능한 S/W 중심 네트워크 기술
	3. 자율제어 기술 : 모니터링을 통한 상태 변화 파악, 자각 기반 상황 인지 기술

- 활용분야 
	1. 스마트 제조
	2. 스마트 교통
	3. 스마트 시티
	4. 스마트 빌딩
	5. 스마트 전력
	6. 스마트 의료
	7. 스마트 농업
