**표현 계층**



응용 프로세스 사이에 전송되는 메시지의 표현 방법을 다룬다.

표현 계층 프로토콜의 전송 메시지에 표현된 문법 내용을 통신 양단의 프로세스가 해석하는

기능을 제공해 송신 프로세스가 전달하는 의미를 수신 프로세스에서 정확히 이해할 수 있게 한다.



**데이터 표현**

응용 환경마다 데이터를 표현하는 방법이 다르기 때문에 다른 문자 체계를 사용하는

컴퓨터끼리 통신하기 위해서는 문자 코트를 변환하는 과정이 필요하다.



**추상 문법과 전송 문법**

컴퓨터마다 사용하는 체계가 다른데 이를 추상문법이라고 하고 중간에 메시지를 전송할 때 문자 체계를 바꿔

전송할 때 사용하는 것이 전송 문법이다.



**ASN.1**

ISO는 분산 환경에서 표현되는 데이터를 정의하기 위한 일반적인 추상 문법으로 ASN.1을 정의했다.



**데이터 압축**

데이터 전송량을 줄이기 위해 압축해서 크기를 줄여서 전송한다.

전송되는 데이터의 크기가 줄어들기 때문에 속도 측면에서 유리하다.

동일한 압축 기법을 사용하더라도 데이터의 형식과 형태에 따라서 압축률이 달라진다.



**-손실 / 비손실 데이터 압축**

말 그대로 압축을 해제했을 때 데이터의 손실이 있으면 손실 압축이고 아니면 비손실 압축이다.
