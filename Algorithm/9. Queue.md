- queue라는 단어 자체가 표를 사기 위해 줄서는 행위를 뜻하며 쉽게 대기열을 의미한다.

stack 구조와 달리 FIFO(First In First Out)형식이다. 대기열을 생각하면 쉽다.

```c++ title='queue'
#include <iostream>  
  
using namespace std;  
  
class queue {  
    public:  
    int arr[10000];  
    int rear;  
    int front;  
  
    queue() {  
        this->rear = 0;  
        this->front = 0;  
    }  
    void push(int val) {  
        this->arr[this->rear] = val;  
        this->rear++;  
    }  
    void pop() {  
        arr[this->front] = -1;  
        this->front++;  
    }  
    int back() {  
        return this->arr[this->rear- 1];  
    }  
    int size() {  
        return this->rear;  
    }  
    bool empty() {  
        return this->rear== 0;  
    }  
};
```

보통 stack은 top 변수를 사용하여 컨테이너의 끝을 알리는 반면, queue는 rear을 통해 알린다.

queue는 스택과 동일한 5가지의 명령함수를 제외하고 한 가지 추가가 되었는데 바로 back()이다.
