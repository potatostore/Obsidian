**ai**: Spring Web 의존성을 추가하면 다음과 같은 기능들을 활용할 수 있습니다.

*   **내장 서버 (Embedded Server)**: Tomcat과 같은 웹 서버가 내장되어 있어 별도의 서버 설정 없이 애플리케이션을 독립적으로 실행할 수 있습니다.
*   **RESTful API 개발**: `@RestController`, `@RequestMapping`, `@GetMapping`, `@PostMapping`, `@PutMapping`, `@PatchMapping`, `@DeleteMapping` 등의 어노테이션을 사용하여 RESTful API를 쉽게 개발할 수 있습니다.
*   **MVC 패턴 지원**: Spring MVC 프레임워크를 통해 Model, View, Controller의 역할을 분리하여 애플리케이션을 구조화하고 유지보수성을 높일 수 있습니다.
*   **JSON/XML 변환 (Jackson 라이브러리)**: Jackson 라이브러리를 통해 Java 객체를 JSON 또는 XML 형식으로 변환(마샬링)하거나, JSON/XML을 Java 객체로 변환(언마샬링)하는 기능을 제공합니다. 이는 REST API 통신에서 데이터를 주고받는 데 필수적입니다.
*   **HTTP 상태 코드 관리**: `ResponseEntity`와 `HttpStatus`를 사용하여 HTTP 요청에 대한 응답 상태 코드를 명확하게 지정하고 관리할 수 있습니다.
*   **애플리케이션 설정**: `@Value` 및 `@ConfigurationProperties` 어노테이션을 사용하여 `application.properties` 파일이나 환경 변수에 정의된 설정을 애플리케이션 내에서 쉽게 사용하고 관리할 수 있습니다.
*   **데이터 접근 추상화 (JPA, Repository)**: JPA(Java Persistence API)를 통해 객체와 데이터베이스 테이블 간의 매핑을 지원하고, Repository 패턴을 통해 데이터 접근 로직을 추상화하여 DB 연동을 간편하게 할 수 있습니다.
*   **자동 설정**: Spring Boot의 자동 설정 기능을 통해 웹 개발에 필요한 다양한 설정을 자동으로 구성해 줍니다.
